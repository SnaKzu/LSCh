Características Principales

Tecnología de vanguardia para reconocimiento de lenguaje de señas

Tiempo Real

Reconocimiento instantáneo de señas con procesamiento a 30 FPS para una experiencia fluida y natural.

IA Avanzada

Red neuronal LSTM entrenada con MediaPipe para detección precisa de pose, rostro y manos.
